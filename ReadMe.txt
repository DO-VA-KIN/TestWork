	Начало работы:

1) Нажмите на кнопку "Конфигурация" и выберете xml файл с описанием кадровой структуры.
(форматы.xml)

2) Нажмите на кнопку "Данные" и выберете cnt или dat файл с данными.
(test.dat - распакованный test.rar)

3) Кнопка "Данные" изменит текст на "Прервать".
 Повторное нажатие на кнопку вызовет отмену фоновой операции.


Верхнее меню "Вид" позволит сокрыть или показать интересующие элементы
-------------------------------------------------------------------------------------------

Прилагается:
*копия тестового задания - "Тестовое задание"
*файл xml с описанием кадровой структуры - "форматы"
*сжатый тестовый файл с данными для проверки - "test"
(необходимо распаковать)

Файл "Созданный" - попытка через serealizer воссоздать xml файл с кадровой структурой 
(пришел к мысли что этот файл формируется не через serealizer.)

Для расчёта CRC использовал табличный метод - больше затрат оперативной памяти, но быстрее.
(что было приоритетным в задании)
